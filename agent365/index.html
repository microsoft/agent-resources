Core capabilities

  • Registry — discover and catalog every agent across the organization
  • Access Control — set policies for which agents can run and who can use them
  • Visualization — dashboards and telemetry across the entire agent fleet
  • Interoperability — works with agents from Microsoft, open-source, and third-party platforms
  • Security — threat detection, compliance, and data protection for agents at scale
